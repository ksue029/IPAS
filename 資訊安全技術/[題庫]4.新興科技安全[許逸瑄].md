```

```

```

```

### 物聯網攻擊模式分析
```

這是屬於哪一類的攻擊手法？ 
(A) 監聽攻擊（Sniffing Attack） (B) 密碼攻擊（Password-Based Attack） 
(C) 金鑰淪陷攻擊（Compromised-Key Attack） (D) 阻斷服務攻擊（Denial-of-Service Attack） 
```
